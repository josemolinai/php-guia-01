# Tabla numérica (índices no ordenados, no consecutivos)

    | Clave/Índice  | Valor    |
    |---------------|----------|
    | 20            | Veinte   |
    | 30            | Treinta  |
    | 10            | Diez     |
    | 5             | Cinco    |

```php
<?php // Tipos de datos Tabla Asociativa ?>
<?php 
    $array = array(
        "0" => "Cero",
        "2" => "Dos",
        "1" => "Uno",
        "3" => "Tres",
    );

// a partir de PHP 5.4
    $array = [
        "0" => "Cero",
        "2" => "Dos",
        "1" => "Uno",
        "3" => "Tres",
    ];
?>
```
